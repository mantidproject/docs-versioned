Table of Contents

New Features¶

  • Background Workspace and Scale Factor can now be given on the ISIS SANS Interface by checking the Scaled Background Subtraction checkbox. See Scaled Background Subtraction for more details.

  • Algorithms SANSTubeCalibration v1 and SANSTubeMerge v1 have been added for calibrating the Sans2d instrument at ISIS.

Bugfixes¶

  • Additional parameter IntegrationRadius in FindCenterOfMassPosition allows reasonable results for asymmetric detector arrays (e.g. BioSANS with midrange and wing detectors)
