Reflectometry¶

Improvements¶

  • LRSubtractAverageBackground now optionally uses weighted error for background with new input property ErrorWeighting.

  • LiquidsReflectometryReduction now optionally uses weighted error for background with new input property ErrorWeighting with the default as False.

  • Version 2 of ConvertToReflectometryQ has been added which assumes detectors are at \(2\theta\) rather than \(\theta_f\). Use version 1 if you require detectors at \(\theta_f\).

  • The detectors positions in the INTER IDF were changed from theta to two theta.
