Reflectometry Changes¶

Table of Contents

  • New Features

  • Bugfixes

New Features¶

  • SaveISISReflectometryORSO v1 now saves the spin state for group workspaces that are subject to polarisation efficiency correction algorithms.

  • New algorithm ReflectometryISISCreateTransmission v1 for creating transmission workspaces that are suitable for passing to PolarizationEfficienciesWildes v1.

Bugfixes¶

  • Polarization Correction and Flood Correction text boxes in the Experiment Settings tab of the Reflectometry Interface are now more robust to file path inputs (e.g. will no longer crash when "C:" is entered).

  • ReflectometryReductionOneLiveData v1 will no longer re-write the spectra map when loading the instrument into the input live data workspace.

  • ReflectometryReductionOneAuto v3 now retrieves the correct exponential correction parameters from the instrument definition file.

  • ApplyFloodWorkspace v1 will no longer modify spectra for which there is no flood correction data in multiple bin correction files.
