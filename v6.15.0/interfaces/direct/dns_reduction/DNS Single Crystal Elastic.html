DNS Single Crystal Elastic¶

The single crystal elastic mode of the DNS Reduction GUI can be used for reduction of polarized and unpolarized single crystal diffraction data collected at the DNS instrument at MLZ.

This mode can be accessed from the main menu of the “DNS Reduction” interface by clicking on “Tools” → “Change Mode” and selecting the “Single Crystal Elastic” option.

Below the interface menu, there are various tabs for data processing. These tabs should be called from left to right during the data processing. The functionality of the “Paths”, “Data”, and “Script Generator” tabs of this operational mode is identical to that of the “Powder TOF” mode.
