Summary¶

Merge the calibrations of the Sans2d front and rear detectors to obtain a single calibration file.

See Also¶

The location to save the merged calibration file to. Allowed values: [‘nxs’]

Description¶

This algorithm merges separate ISIS Sans2d front and rear detector calibration files that have previously been created from SANSTubeCalibration v1. A single workspace is created with an empty instrument containing the calibrated positions of both sets of detector pixels. The final workspace will be saved out as a Nexus file to the location specified in the OutputFile property, if one has been provided.
