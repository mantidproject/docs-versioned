Frequency Domain Analysis¶

Bugfixes¶

  • The Muon FDA interface will no longer display an unnecessary error prompt after cancelling a MaxEnt calculation.

Muon Analysis¶

Bugfixes¶

  • External plots will no longer be squashed by plot legends.

ALC¶

New features¶

  • The Muon ALC interface can now import data from an externally generated matrix workspace (i.e. not previously exported from the Muon ALC interface).

Algorithms¶

New features¶

  • The LoadMuonNexus v2 algorithm no longer chooses a different Muon loader if appropriate. This functionality has been moved to a new algorithm version, LoadMuonNexus v3.

  • Users can also call the Load algorithm directly which will also select the appropriate loader for them.
