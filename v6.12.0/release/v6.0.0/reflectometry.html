ISIS Reflectometry Interface¶

  • Runs in the search results list can now be excluded or annotated, e.g. to exclude runs from autoprocessing. See the ISIS Reflectometry Interface documentation for details.

  • A big has been fixed where the flood workspace drop-down box was being accidentally populated with the first workspace created after clearing the workspaces list.

Algorithms¶

  • A bug has been fixed where import CaChannel on Linux would cause a hard crash.

  • A bug has been fixed where running ReflectometryReductionOneAuto v3 on workspace groups resulted in spurious error messages about transmission workspaces.
