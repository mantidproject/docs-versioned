DNS Powder TOF¶

This mode can be used to perform reduction of inelastic (time of flight) data for powders. It can be accessed from the main menu of the “DNS Reduction” GUI by clicking on “Tools” → “Change Mode” and selecting the “Powder TOF” option.

Below the interface menu, there are various tabs for data processing. These tabs should be called from left to right during the data processing.
