Description¶

This is a workflow algorithm that creates files suitable as input into PDFgetN.

  1. PDLoadCharacterizations v1

  2. LoadEventAndCompress v1 if InputWorkspace is not provided

  3. PDDetermineCharacterizations v1 to determine information from the characterization file

  4. AlignAndFocusPowder v1

  5. NormaliseByCurrent v1

  6. SetUncertainties v1 (SetError="sqrt")

  7. SaveGSS v1
