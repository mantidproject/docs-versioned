Interface Overview¶

This interface is used for visualizing cuts and slices of inelastic neutron scattering data. The interface can be accessed from the main menu of MantidWorkbench, in Interfaces → Direct → MSlice.

MSlice Standalone¶

When MSlice is installed as a separate conda package, it can be opened with the command mslice

Documentation¶

The current MSlice documentation can be viewed at

http://mantidproject.github.io/mslice
