Muon Analysis Plotting Window¶

The Muon Analysis GUI has some extra plotting functionality.

Fit Data¶

This is shown when the user selects the following tabs:

  • Fitting

  • Sequential Fitting

  • Results

It displays the selected data for a fit and if a fit is requested this is also shown on the plot. In the sequential fitting tab, the user can select the data to plot by highlighting the different rows in the table.
