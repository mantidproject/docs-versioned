.. _func-FullprofPolynomial:

==================
FullprofPolynomial
==================

.. index:: FullprofPolynomial

Description
-----------

A fullprof polynomial function of degree :math:`n` is defined as:

.. math:: y = A_0 + A_1 \times x + A_2 \times x^2 + ... + A_n \times x^n, \text{ where } x = \frac{\text{tof}}{\text{Bkpos}}-1

and :math:`A_0, ...,  A_n` are constant coefficients. Note: :math:`n` must be either 6 or 12.
