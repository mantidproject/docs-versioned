Reflectometry Changes¶

Table of Contents

  • New Features

  • Bugfixes

New Features¶

  • New algorithm PolarizationEfficienciesWildes v1 has been added for calculating the efficiencies of the polarizing components of an instrument with two flippers. This algorithm implements the approach from the A. R. Wildes 2006 paper doi: 10.1080/10448630600668738.

Bugfixes¶

  • Stitch1DMany algorithm now runs without errors via the dialog for executing algorithms.

  • The orsopy library has been updated to version 1.2.1. This pulls in a bug fix where multi-dataset reduced reflectivity .ort files with different column headers for different datasets would use only the headers for the first dataset throughout the file.

  • Algorithm ReflectometryReductionOneLiveData v1 now applies polarization corrections correctly on WorkspaceGroup inputs.

  • When on IDAaaS, only runs that are available in the ISIS Instrument Data Cache will now appear in the Search Runs section of the Runs tab when the Search Data Archive setting is set to Off.
